# Portfolio Summary

## Project

Vehicle Insurance Data Model

## Short CV Description

Designed a complete vehicle insurance database model from conceptual design to logical and physical models, then generated Oracle Database 21c DDL using Oracle SQL Developer Data Modeler.

## Key Points

- Built a model covering customers, accounts, quotes, underwriting, policies, drivers, vehicles, invoices, payments, claims, renewals, endorsements, telematics, and no-claim bonus tracking.
- Developed CDM, LDM, and PDM stages with Oracle SQL Developer Data Modeler files included.
- Produced a final physical model with 26 tables, 32 foreign keys, primary keys for all tables, and unique constraints for important identifiers.
- Applied normalisation by separating customer/account data, driver/licence/penalty data, policy-driver relationships, and payment subtypes.
- Documented the main design decisions, model changes, error fixes, and final results.
